### 功能
1.生成钱包，查询ETH主网ETH和USDT余额、BSC链BNB和USDT余额<br />
2.写入数据库有余额的钱包私钥<br />
3.尝试转出钱包所有余额到指定地址<br />

### 注意😀
1.需修改./db/dbConfig.js中你的DB信息，./config/config.js中collectAddress<br />
2.启动例如：env MYSQL_PASSWD=xxxx node index.js，当然，你也可以创建修改.env或写入你的package.json
